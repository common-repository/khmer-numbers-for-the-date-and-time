== Description ==

This plugin translates all numbers for dates and comments into Khmer.  You can use this plugin with the Khmer language pack (http://km.wordpress.org/). Created by the Society for Better Books in Cambodia (http://www.sbbic.org)/.

== Installation ==

1. Upload `Khmer Numbers for the Time and Date.zip` to the `/wp-content/plugins/` directory and extract the zipped file.
2. Activate the plugin through the 'Plugins' menu in WordPress
